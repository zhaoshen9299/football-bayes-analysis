# 经验验证记录：英超 2025/26

## 研究设计

- 执行日期：2026-07-14。
- 训练／预热：英超 2023/24，380场。
- 参数选择：英超 2024/25，380场。
- 最终留出测试：英超 2025/26，380场；测试季没有用于调参。
- 结果、角球和赔率来自 Football-Data 历史 CSV；市场基准使用多机构平均开盘与收盘赔率分别比例去水。
- 同一开球时间先全部预测，再批量更新历史。该规则防止末轮同时开球数据泄漏。
- 收盘赔率仅代表临近开赛的强基准，不能代表更早时间点可执行的预测。

## 锁定参数

验证集选择的进球参数为先验权重5、半衰期365天、Dixon-Coles `rho=-0.10`；角球为先验权重40、半衰期180天。验证集为模型—市场混合选择了市场权重1.0，因此最终混合结果等于收盘市场；这说明当前独立模型没有获得可验证的增量权重。

模型减收盘市场的逐场 Log Loss 差为 `+0.018970`，95% Bootstrap 区间 `[-0.001352, +0.038700]`；点估计较差，但区间跨0，不能宣称差异稳定。开盘减收盘为 `+0.003486`，区间 `[-0.002712, +0.009760]`，同样不宜夸大。

角球模型总数 MAE 为2.635824，朴素联赛均值基线为2.650720；模型减基线为 `-0.014897`，95%区间 `[-0.042424, +0.013649]`。改善很小且区间跨0。总角球均值误差为 `+0.202518`，90%预测区间覆盖率92.63%。

## 可迁移经验

1. 市场必须作为正式基准，混合权重必须在未见验证集选择，不能预设“模型加市场一定更好”。
2. 单季点估计不足以证明优势；报告逐场配对区间，并把“区间跨0”写进结论。
3. 英超平局率从2023/24的21.58%升至2025/26的27.37%，总进球从3.279降至2.750，显示赛季基线会漂移。应加入赛季层级或动态基线监控。
4. `rho=-0.10` 缩小了低比分独立模型的平局低估，但仍需在以后赛季重新验证，不可外推为通用参数。
5. 角球总量由2023/24的10.837降至2025/26的9.997。球队风格极端和升班马会产生系统残差，后续应验证球队随机效应、负二项离散和比赛状态协变量。
6. 本记录是一次历史留出研究，不等于当时真实冻结的逐日赛前发布记录，也不证明投注收益。

## 当前限制

- 验证调参以角色后验均值构造概率，测试则用 Monte Carlo 传播参数不确定性；下一版应让调参与最终预测管线完全同构。
- 当前测试使用固定的上一验证季联赛基线，只审计跨季漂移，尚未实现测试季内动态基线。
- Bootstrap 按比赛独立重采样，未按比赛日或同开球批次聚类；区间可能低估共同冲击造成的不确定性。
